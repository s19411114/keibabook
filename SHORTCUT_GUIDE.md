# ショートカット作成ガイド

Streamlitアプリを簡単に起動するためのショートカット作成方法です。

## 方法1: バッチファイルのショートカット（推奨）

### ステップ1: ショートカットを作成

1. `start_streamlit.bat` を右クリック
2. 「ショートカットの作成」を選択
3. 作成されたショートカットを右クリック → 「プロパティ」
4. 「アイコンの変更」でアイコンを設定（オプション）
5. ショートカットをデスクトップまたはタスクバーにドラッグ

### ステップ2: タスクバーにピン留め

1. ショートカットを右クリック
2. 「タスクバーにピン留め」を選択

## 方法2: PowerShellスクリプトのショートカット

1. `start_streamlit_powershell.ps1` を右クリック
2. 「ショートカットの作成」を選択
3. ショートカットを右クリック → 「プロパティ」
4. 「リンク先」を以下のように変更：
   ```
   powershell.exe -ExecutionPolicy Bypass -File "C:\GeminiCLI\TEST\keibabook\start_streamlit_powershell.ps1"
   ```
5. デスクトップやタスクバーに配置

## 方法3: 直接WSLコマンドのショートカット

1. デスクトップで右クリック → 「新規作成」→ 「ショートカット」
2. 以下のコマンドを入力：
   ```
   wsl bash -c "cd /mnt/c/GeminiCLI/TEST/keibabook && source venv/bin/activate && streamlit run app.py"
   ```
3. ショートカット名を「競馬ブックスクレイパー」などに設定
4. 完了

## アイコンの設定（オプション）

ショートカットのアイコンを変更する場合：

1. ショートカットを右クリック → 「プロパティ」
2. 「ショートカット」タブ → 「アイコンの変更」
3. お好みのアイコンを選択（🐎 や ⚡ など）

## トラブルシューティング

### WSLが見つからないエラー
- WSLがインストールされているか確認
- `wsl --version` で確認

### 仮想環境が見つからない
- まず `SETUP.md` の手順で仮想環境を作成してください

### ブラウザが自動で開かない
- 手動で `http://localhost:8501` にアクセスしてください
